# Marketplace Statuses

## Publisher Website Statuses

| Status | Meaning |
| --- | --- |
| `pending_review` | Submitted by publisher, waiting for admin |
| `approved` | Visible in marketplace |
| `rejected` | Not visible in marketplace |

## Guest Post Order Statuses

| Status | Meaning |
| --- | --- |
| `pending_publisher` | Advertiser bought placement; publisher needs to publish |
| `submitted` | Publisher submitted live article URL |
| `completed` | Fulfillment approved and payout released |

## Future Statuses

Likely future additions:

- `revision_requested`
- `cancelled`
- `disputed`
- `refunded`
