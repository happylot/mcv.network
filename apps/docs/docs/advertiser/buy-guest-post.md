# Buy Guest Post

Advertisers buy guest posts from approved publisher websites in `/marketplace/websites`.

## Required Fields

| Field | Description |
| --- | --- |
| Target URL | URL the article should link to |
| Anchor text | Optional preferred anchor |
| Article title | Optional suggested title |
| Requirements | Topic, tone, link rules, or editorial notes |

## Purchase Rules

- Website must be `approved`.
- Advertiser cannot buy from their own website.
- Wallet must have enough available balance.

## After Purchase

The order status becomes `pending_publisher`.

The advertiser can track orders in `/marketplace/orders`.

When the publisher submits a live article URL, status becomes `submitted`.

The advertiser can approve the order to release payout.
