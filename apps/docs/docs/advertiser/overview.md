# Advertiser Overview

Advertisers use `ads.mcv.network` to fund their wallet and buy guest post placements from approved publisher websites.

## Main Navigation

Advertisers see:

- Dashboard
- Marketplace
- Guest Post Orders
- Add Funds
- Activity Log

## Standard Workflow

1. Log in to `ads.mcv.network`.
2. Add wallet funds through Stripe Checkout.
3. Open Marketplace.
4. Choose an approved publisher website.
5. Submit target URL, anchor text, article title, and content requirements.
6. Wait for the publisher to submit the live article URL.
7. Review the published URL.
8. Approve fulfillment to release payout to publisher.

## Wallet Requirement

Guest post purchase is prepaid.

If wallet balance is lower than the website price, the portal redirects the advertiser to Billing to add funds.
