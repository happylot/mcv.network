# Admin Overview

Admins operate marketplace quality and fulfillment.

## Main Navigation

Admins see:

- Dashboard
- Website Review
- Order Review
- Marketplace Ops
- Disputes
- Payout Review

## Current Admin Jobs

- Approve or reject publisher websites.
- Review submitted guest post fulfillments.
- Approve orders to release publisher payout.
- Monitor marketplace status counts.

## Admin Account

An admin user needs an account with:

```text
accounts.type = admin
account_user.role = owner
```
