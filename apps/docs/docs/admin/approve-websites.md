# Approve Publisher Websites

Admins review publisher website submissions from `/admin/publisher-websites`.

## Review Checklist

Before approving:

- Domain is real and appropriate.
- Niche matches the website content.
- DR/DA/traffic are plausible.
- Price is reasonable.
- Guidelines are clear enough for advertisers.
- Sample URL is acceptable when provided.

## Actions

| Action | Result |
| --- | --- |
| Approve | Website status becomes `approved` and appears in marketplace |
| Reject | Website status becomes `rejected` and does not appear in marketplace |

## Marketplace Visibility

Only websites with this status are visible:

```text
approved
```
