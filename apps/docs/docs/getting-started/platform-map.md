# Platform Map

## Public Marketing

Domain: `mcv.network`

Purpose:

- Marketing pages.
- SEO landing pages.
- Content and blog when WordPress is installed.
- Login and signup CTAs should point to `ads.mcv.network`.

Important links:

- Login CTA: `https://ads.mcv.network/login`
- Signup CTA: `https://ads.mcv.network/signup`

## Advertiser Portal

Domain: `ads.mcv.network`

Purpose:

- Advertiser and publisher authentication.
- Wallet top-up.
- Guest post marketplace.
- Publisher inventory and order fulfillment.
- Admin approval workflows.

## Documentation

Domain: `docs.mcv.network`

Purpose:

- User guides.
- Admin runbooks.
- Deployment notes.
- Release notes.

Implementation:

- Docusaurus under `apps/docs`.
- Markdown-driven docs in Git.
- Static build deployed by GitHub Actions.
