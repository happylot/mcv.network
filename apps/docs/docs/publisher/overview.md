# Publisher Overview

Publishers use `ads.mcv.network` to list websites for guest post sales and fulfill advertiser orders.

## Main Navigation

Publishers see:

- Dashboard
- My Websites
- Add Website
- Guest Post Orders
- Payouts

## Standard Workflow

1. Log in as a publisher.
2. Submit a website.
3. Wait for admin approval.
4. Receive guest post orders from advertisers.
5. Publish the article.
6. Submit the live published URL.
7. Wait for advertiser or admin approval.
8. Receive payout in wallet.

## Payout Timing

Funds are released to the publisher only after the order is approved.

Approval can be done by:

- The advertiser who bought the placement.
- An admin, when manual intervention is needed.
